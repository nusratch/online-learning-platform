SkillSphere 🎓




About

SkillSphere is a simple online learning platform where users can explore different courses, see details, and manage their profile. It is built with a clean design and works smoothly on all devices.


Features
- User registration and login
- Browse all courses
- Search courses by name
- View course details (protected)
- Profile page with update option
- Toast notifications for actions
- Responsive design (mobile, tablet, desktop)


Tech Used
- Next.js
- React
- Tailwind CSS
- MongoDB Atlas
- React Toastify
- Framer Motion
- BetterAuth


Packages
- react-toastify
- framer-motion
- mongodb
